# phpBB 3.1 Extension - Posts count

## License
[GPLv2](license.txt)

### Requirements
- phpBB 3.1.3 or higher
- PHP 5.3.3 or higher

#### Languages supported
- English
- German

##### Installation
1. Download the archive, unzip it.
2. Copy the folder kirk into root/ext
3. Navigate into the ACP to Customise -> Manage extensions.
4. Go to "ACP" > "Customise" > "Extensions" and enable the "Posts count" extension.
5. Clear all caches
